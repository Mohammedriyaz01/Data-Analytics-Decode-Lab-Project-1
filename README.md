# 📊 Data Analytics - Decode Labs Project 1

## 📌 Project Overview

This project was completed as part of the **Decode Labs Data Analytics Internship**.

The objective was to clean and prepare a raw dataset by handling missing values, duplicate records, and formatting inconsistencies to make it ready for analysis.

---

## 🎯 Objectives

- Identify missing values
- Handle null values appropriately
- Remove duplicate records
- Standardize data formats
- Prepare a clean dataset for analysis

---

## 🛠️ Technologies Used

- Python
- Pandas
- Jupyter Notebook

## ✅ Data Cleaning Tasks Performed

- Checked dataset structure using `df.info()`
- Identified missing values
- Filled missing `CouponCode` values
- Verified duplicate records
- Converted `Date` column to datetime format
- Verified data types
- Exported cleaned dataset
